Para compilar el programa en la consola (si se tiene el compilador de C instalado) cambiar al directorio que tiene a los archivos

Poner: gcc proyecto.c escritura.c inercia.c

Si se quiere poner un nombre poner -o nombreDelPrograma

Correrlo desde el mismo directorio con ./nombreDelPrograma

Si no se le puso un nombre el default es "a" por lo que se pondria ./a
